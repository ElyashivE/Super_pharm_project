-------
האתר משמש כאתר מסחר לרכישות / וחיפוש מוצרים באתר

הבדיקות שלי:
---------------

הסבר מבנה:
--------------
מבנה הפרוייקט בנוי לפי מבנה כתיבת ה STD לכל class יש נושא משלו המתמקד בבדיקות עבור אותו נושא.
- ישנם סך הכל 11 classes של testNG המייצרים כ 30 טסטים כפי שנידרש בהכנה לפרוייקט תחת Package Testing.
- ישנו class נוסף המייצר ממשק Jframe לנוחות הרצת הבדיקות הנמצא גם תחת Package Testing. 
- ישנו class נוסף המשמש לייצור extent report.
- ישנו class נוסף המשמש לפונקציות כלליות.
שני classes אלו נמצאים תחת Package Assists.
